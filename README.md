# MMC是什么? 
## 简介
MMC是一个使用Python编写的快速模组(Modifcations)生成器, 开源, 使用方便。
## 优点
MMC 可以快速生成对于Minecraft JE的模组代码。暂不支持GUI。
## 支持版本
Minecraft Java Edition

Fabric 1.20.1

Forge 暂不支持

注：Forge 参见 MCreator <https://mcreator.net/>
